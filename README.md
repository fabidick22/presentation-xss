# presentation-xss
Presentation with reveal.js about xss 

# Empezar presentación
Librerias completas
```sh
git clone https://github.com/fabidick22/presentation-xss.git
cd presentation-xss/reveal.js/
npm install
npm start
```
Solo lectura
```sh
git clone https://github.com/fabidick22/presentation-xss.git
cd presentation-xss/reveal.js/
firefox index.html # abrir idex.html
```
